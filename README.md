# fbneo-kof2003-framedata

This project is a **specialized fork of the FBNeo Lua training mode**, focused **exclusively on *The King of Fighters 2003***.

The main goal of this fork is to **add advanced framedata and internal combat values that were not available in the original script**, turning it into a powerful analysis and training tool for KOF 2003.

While the original FBNeo training mode supports many fighting games with general training features, **this fork is fully dedicated to KOF 2003** and significantly expands the amount of real-time combat data available to the player.

This script is designed for players who want to **understand the game at a deeper, technical level**, including frame advantage, stun values, and move properties.

---

## ðŸ†• KOF 2003 Exclusive Features

This fork adds **advanced framedata visualization**, including:

- **Startup Frames**  
  Number of frames before an attack becomes active.

- **Frame Advantage on Hit**  
  Advantage or disadvantage after a successful hit.

- **Frame Advantage on Block**  
  Advantage or disadvantage when the attack is blocked.

- **Guard Stun Values**  
  Exact guard stun duration applied to the opponent.

- **Stun Values**  
  Exact hit stun duration on hit.
